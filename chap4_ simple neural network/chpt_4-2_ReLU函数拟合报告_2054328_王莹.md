## 1. 函数定义

本实验目标是使用多层感知机（MLP）拟合目标函数 $y=\sin(x)$。

### 1.1 目标函数

```python
def target_function(x):
    """目标函数：sin(x)"""
    return np.sin(x)
```

### 1.2 前向计算与训练流程

本次实现未单独封装 `train()`、`test()` 函数，而是在主流程中完成训练与测试。核心步骤如下：

1. 前向传播：`y_pred = model(x_train_t)`
2. 计算损失：`loss = loss_fn(y_pred, y_train_t)`
3. 梯度清零：`optimizer.zero_grad()`
4. 反向传播：`loss.backward()`
5. 参数更新：`optimizer.step()`

每 100 个 epoch 输出一次训练损失，便于观察收敛情况。

## 2. 数据采集

本实验在区间 $[-\pi,\pi]$ 上构造样本并拟合正弦曲线。

### 2.1 数据生成方式

- 训练集：在 $[-\pi,\pi]$ 随机采样 300 个点
- 测试集：在 $[-\pi,\pi]$ 均匀采样 100 个点
- 随机种子：`np.random.seed(42)`（保证可复现）

对应代码如下：

```python
np.random.seed(42)

x_train = np.random.uniform(-np.pi, np.pi, 300).reshape(-1, 1).astype(np.float32)
y_train = target_function(x_train).astype(np.float32)

x_test = np.linspace(-np.pi, np.pi, 100).reshape(-1, 1).astype(np.float32)
y_test = target_function(x_test).astype(np.float32)
```

数据维度：

- 训练集：`(300, 1)`
- 测试集：`(100, 1)`

## 3. 模型描述

按照题目“ReLU函数拟合”的要求，采用 3 个线性层和 2 个 ReLU 激活层构建网络。

### 3.1 网络结构

```python
class NeuralNet(nn.Module):
    def __init__(self):
        super(NeuralNet, self).__init__()
        self.layer1 = nn.Linear(1, 64)
        self.layer2 = nn.Linear(64, 32)
        self.output = nn.Linear(32, 1)
        self.relu = nn.ReLU()

    def forward(self, x):
        x = self.relu(self.layer1(x))
        x = self.relu(self.layer2(x))
        x = self.output(x)
        return x
```

结构说明：

- 输入层：1 维（标量 $x$）
- 隐藏层1：64 个神经元 + ReLU
- 隐藏层2：32 个神经元 + ReLU
- 输出层：1 维回归输出（预测 $\sin(x)$）

### 3.2 损失函数与优化器

```python
model = NeuralNet()
optimizer = torch.optim.Adam(model.parameters(), lr=0.01)
loss_fn = nn.MSELoss()
```

- 损失函数：均方误差（MSE）
- 优化器：Adam
- 学习率：0.01
- 训练轮数：500

## 4. 拟合效果

训练过程中损失稳定下降，模型对正弦曲线有较好拟合能力。测试阶段计算均方误差：

with torch.no_grad():

## 5. 实验结论

1. 使用 ReLU 激活的三层全连接网络可以有效拟合一维非线性函数 $\sin(x)$。
2. 在当前参数设置下（300 训练样本、500 轮训练、Adam 优化），模型具有较好的收敛性和泛化能力。
